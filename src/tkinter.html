Note that:

Text and Button like widget treat width and height options as characters while Frame , canvas like widgets treat it as pixels
Basically , among text and frame of width 100 ,text appears to be of larger width than frame
Same with height

Menu Widget in Tkinter

Creating menu widget instance with options

myMenu = Menu(root,tearoff=0)
# Setting tearoff=0 will remove ugly borders that appears on the widget

Adding the Menu to the Main Window

root/master.config(menu=myMenu)

Add Cascade Function Of menu widget

    CODE
    DESC
myMenu.add_cascade(label="Name of menu"
, menu = menuObject)
# Use Multiple times to add Multiple menus
Adds another Menu widget instance to the myMenu instance
Basically adds another menu in the main myMenu .

Add Command Function of menu widget

    CODE
    DESC
# Let us give commands to file menu shown in the image
menuBar.add_command(label="Close"
, command=exit)
# Use Multiple Commands to create Multiple Options
This function is used to create options in vertical direction in menu (in this case file) If user clicks the option the command will be executed

What happens when you use add_command with myMenu and not with child menu

Adding a separator between the menu Options

syntax: menuInstance.add_separator()
